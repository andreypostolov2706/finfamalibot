# FinFamily Bot - Телеграм бот для учёта финансов

Телеграм бот с ИИ-ассистентом (DeepSeek) для учёта личных и бизнес-финансов.

## Установка

### 1. Клонирование репозитория
```bash
git clone https://github.com/andreypostolov2706/AIFullbot.git
cd FinFamale1
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
```

### 3. Активация виртуального окружения

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 4. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 5. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env`:
```bash
copy .env.example .env
```

Отредактируйте `.env` и укажите:
- `BOT_TOKEN` - токен вашего Telegram бота от @BotFather
- `DEEPSEEK_API_KEY` - API ключ DeepSeek
- `ADMIN_IDS` - ваш Telegram ID (можно узнать у @userinfobot)

### 6. Запуск бота
```bash
python main.py
```

## Основные функции

### Семейный бюджет
- Добавление расходов через естественный язык: "100 рублей шоколадка"
- Автоматическая категоризация через ИИ
- Обработка чеков с OCR

### Бизнес-аккаунты
- Отдельный бизнес для каждого пользователя
- Учёт доходов и расходов
- Выдача зарплаты с автоматическим отчислением в копилку

### Копилки
- Автоматическая копилка "Шекель 10%" (10% от зарплаты)
- Пользовательские копилки ("На море", "На машину")

### Управление кредитами
- Добавление фиксированных платежей
- Отображение в дашборде

### Статистика
- Детальная статистика по категориям и подкатегориям
- Отчёты за месяц

## Команды бота

- `/start` - Регистрация/вход
- `/help` - Справка по командам
